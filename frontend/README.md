# CareerAI - React Frontend

AI-Powered Career Guidance Platform built with React 18.x, React Router, and Material-UI.

## Tech Stack

- **Framework**: React 18.x with Vite
- **Routing**: React Router v6
- **State Management**: React Context API
- **UI Library**: Material-UI (MUI) v5
- **Charts**: Recharts
- **Icons**: Lucide React
- **HTTP Client**: Axios

## Installation

```bash
# Navigate to the frontend-react directory
cd frontend-react

# Install dependencies
npm install

# Start development server
npm run dev
```

## Available Scripts

- `npm run dev` - Start development server (default: http://localhost:5173)
- `npm run build` - Build for production
- `npm run preview` - Preview production build

## Environment Setup

Make sure your backend API is running on `http://127.0.0.1:8000`
